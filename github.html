GitHub







.Por que usar?


O GitHub é uma plataforma de hospedagem de código fonte e repositório de projetos de código aberto, essencial para os desenvolvedores em todo o mundo. Ele permite que os desenvolvedores colaborem em projetos de software, gerenciem versões de código, compartilhem seu trabalho com outros desenvolvedores e se conectem com a comunidade de desenvolvimento. Com ferramentas de controle de versão, rastreamento de problemas e outras ferramentas de colaboração, o GitHub é uma ferramenta valiosa para equipes de desenvolvimento geograficamente distribuídas e indivíduos que desejam construir um portfólio de trabalho e contribuir para a comunidade de desenvolvimento de software.





.Repositórios


Um repositório no GitHub é uma pasta virtual que armazena o código fonte de um projeto de software. Ele é usado por desenvolvedores para compartilhar e colaborar em projetos de software.

O repositório no GitHub inclui recursos avançados, como ferramentas de controle de versão, que permitem que os desenvolvedores gerenciem as alterações feitas em seu código ao longo do tempo. O uso de repositórios no GitHub é fundamental para o processo de desenvolvimento de software moderno e altamente colaborativo, permitindo que os desenvolvedores trabalhem em equipe, gerenciem versões de código e facilitem a colaboração.



.Criando um repositorio


Passo a passo:

Em seu GitHub, expanda o ícone do seu perfili e clique em "Your repositories"

Na página com seus repositórios clique no botão verde "new"

Na aba de criar um novo repositório a configuração é simples:

1. Coloque o nome do repositório;

2. Uma descrição(opcional);

3. Defina se ele será Público ou Privado;

4. Marque a opçaõ adicionar arquivo "Read.me"(indicado);

5. Agora clique em "Create repository";

Pronto! Agora seu repositório está criado e você pode enviar seus arquivos





.Enviando um arquivo para o GitHub


Existem algumas formas de se enviar um arquivo para seu repositório, a que vamos ensinar é usando o terminal de comando Git, aliás é necessário instalar o git em seu computador. Para isso deixaremos 2 links abaixo que vão te ajudar:

Instalação do Git
Entendendo o Git e GitHub

1. Primeiramente crie uma pasta em seu computador destinada aos seus projetos

2. Com o Git instalado, vá até sua pasta, clique com o botão direito do mouse e inicie o "Git Bash Here"

3. Com o teminal aberto vamos clonar seu repositório. Digite: git clone "caminho do seu repositório-sem aspas"

4. Para "pegar" o caminho do seu repositório, vá até seu github e no seu repositório copie o link como na imagem abaixo e depois cole com o atalho (Shift + Insert) no seu terminal Git.



5. Agora será criado outra pasta automáticamente dentro da sua pasta. Será dentro desta nova pasta que vovcê executará o terminal do Git e subir os arquivos para seu repositório.
